## 🚀 核心成就

### 1. AI 協作與工程流優化

在遠傳期間，我最顯著的成就之一是推動了 Spec-Driven Development (SDD) 流程：

- 效率提升：利用 GitHub Spec-Kit 與 AI 工具鏈(markitdown)，有效解決AI失憶與幻覺(有效減少context長度)，將規格文件產出時間由 4 小時縮短至 30 分鐘（提升 88%）。

### 2. 電子發票與權限治理

主導企業級權限管理平台 & 電子發票的架構設計與實作：

- 技術整合：深度整合 Keycloak 與 SSO 登入流程。
- 效能優化：設計覆蓋 100 支以上 API 的權限映射機制，使新功能接入權限系統的時間由 5 天縮短至 1 天（縮短 80%）。

### 3. Cloud Native & DevOps

建立 Kubernetes 環境的標準化部署流程：
